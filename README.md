## CAPDE

#### O que é o CAPDE?

> O CAPDE é um software de código aberto, desenvolvido com o intuito de auxiliar no processo de gerenciamento dos servidores, de modo que, facilite o cadastro, capacitação e identificação de servidores devidamente aptos para colaborar em determinada região.
> O software é distribuído sob MIT License - qual pode ser conferida em [LICENSE](https://github.com/wesleyabelini/CAPDE/blob/master/LICENSE).


#### Quick Start

> Acesse a nossa [Wiki](https://github.com/wesleyabelini/CAPDE/wiki), e encontre em nossos guias informações que o ajudarão com os requisitos necessários para a instalação, bem como downloads e demais configurações.
> Em [Releases](https://github.com/wesleyabelini/CAPDE/releases) é possível encontrar versões do CAPDE disponíveis para download.
